How to use "./YelpDatabaseProject/Parser.cs":
This file contains the code necessary to parse JSON files and output to a given file.
To exclude certain types of data you can either give the constructor a list of strings of the types or use the ChangeExceptions() method.
The Convert() method takes in an input file and an output file which will convert the input file to the output file.

The rest of this directory is the sample application for pulling and displaying information for the UI.
